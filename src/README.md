# CTF
* 每个人新建自己的目录以自己名字，命名文件夹
* 再在自己的目录下面命名具体题目的名字、里面放writeup（可以写到我们的知乎，添加链接到这里），和题目文件

```
CTF/

	README.md
	
	题目1:/
	
	题目2:/
```

# SRC
* 每个人新建自己的目录以自己名字，命名文件夹
* 新建README.md 具体描述一下自己做的内容
* 举例：2019年10月10号补天提交SQL注入漏洞

```
SRC/

	README.md
```

# 如果CTF和SRC都参加
* 目录结构

```
Your Name/

	CTF/
	
		README.md
		
		题目1:/
		
		题目2:/
		
	SRC/
	
		README.md
```
